# plantoid

How to test :
 - install [Truffle](https://github.com/ConsenSys/truffle) and initialize your project with `truffle init`.
    ```sh
    npm install -g truffle
    mkdir myproject && cd myproject
    truffle init
    ```
 - install packages 
    ```sh
    npm install 
    ```
- run ganache 
    ```sh
    npm run ganache 
    ```
 
 - Compile migrate and test
   ```sh
    truffle compile 
    truffle migrate
    truffle test
    ```
